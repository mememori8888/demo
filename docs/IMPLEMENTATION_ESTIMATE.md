## 📌 重要事項

### システム利用前提条件

- GitHub Actionsの実行環境が利用可能
- BrightData APIのアカウント
- 基本的なGitHub操作知識（設定のみ）

### サービス保証範囲

**保証内容**
- システムの正常動作
- エラー時の自動リトライ
- データ品質の基本検証

**保証外**
- BrightData APIの障害
- GitHub Actionsの障害
- Googleマップ側の仕様変更
- 取得データの内容精度（Googleマップ依存）

### データ利用上の注意

- 取得データはGoogleマップの公開情報
- 商用利用時はGoogleの利用規約を確認
- 個人情報の取り扱いに注意
