## 📊 データ統合の概要

### データソース

- **extractedシート（WordPress元データ）**: 69,742件のPost ID

### 統合方法

施設名・住所・電話番号を用いて、WordPressのPost IDとGoogle Mapsの施設データを照合

## 🔄 提案　今後の定期的なデータチェック計画

### 監視項目

**1. Post ID整合性チェック**
- WordPress extractedシートの新規Post ID追加を検出
- integrated_data_updatedシートとの差分を確認
- 未統合Post IDの早期発見

**2. 施設名変化チェック（全検索）**
- **既存62,807件全施設をBrightData API経由で再検索**
- 施設名の変更を検出（例：リニューアル、名称変更、移転）
- Google Maps上の最新情報と比較・更新

**3. 営業ステータス更新**
- 営業中施設の閉業検出
- 一時休業から営業再開の検出
- 営業時間の変更確認

### 実装方針

| 項目 | 頻度 | 対象件数 | 説明 |
|------|------|----------|------|
| Post ID比較 | 月次 | 全Post ID | 新規追加検出 |
| **全施設名検索** | **月次** | **62,807件** | **施設名変化チェック** |
| ステータス更新 | 月次 | 62,807件 | 営業状態確認 |

### 想定される変化

- 📝 施設名変更（リニューアル、ブランド変更）
- 📍 住所変更（移転）
- 📞 電話番号変更
- 🏢 営業ステータス変化（営業中→閉業、一時休業→営業再開）

**定期的なチェックにより、Google Mapsの最新情報を常に反映した施設データベースを維持します。**
