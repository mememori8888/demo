# GoogleMAPデータ取得統合ガイド
> 業種別データ監視システム - 運用マニュアル

**最終更新:** 2026年02月06日 | **バージョン:** 2.1

---

## システム概要

### 2つの取得モード

| モード | 対象業種 | 取得データ | 前提条件 | 月額コスト |
|:---|:---|:---|:---|:---|
| **レビュー監視** | 歯科医院、結婚相談所 | 口コミレビュー | 施設リスト保有 | 100円〜1,500円 |
| **施設情報収集** | 介護施設 | 施設基本情報 | なし | $10〜$45（初回） |

### 選択フロー

```
保有データ確認
├─ 施設リスト有 → レビュー監視モード
│   └─ ワークフロー: brightdata_reviews_新仕様
└─ 施設リスト無 → 施設情報収集モード
    └─ ワークフロー: brightdata_facility
```

## 実行手順

### レビュー監視モード

**対象**: 歯科医院、結婚相談所  
**ワークフロー**: `brightdata_reviews_新仕様.yml`

**実行パス**: `GitHub Actions` → `Actions` → `brightdata_reviews_新仕様`

**必須パラメータ**:

| パラメータ | 設定例 | 説明 |
| :--- | :--- | :--- |
| csv_file | `results/dental_new_hokkaido.csv` | 施設データCSV |
| output_file | `results/dental_reviews_hokkaido.csv` | レビュー出力先 |
| days_back | `10` | 取得期間（日数） |
| skip_column | `GoogleMap` | スキップ判定列 |

**処理時間**: 30分〜3時間（データ量依存）

---

### 施設情報収集モード

**対象**: 介護施設  
**ワークフロー**: `brightdata_facility.yml`

**実行パス**: `GitHub Actions` → `Actions` → `brightdata_facility`

**必須パラメータ**:

| パラメータ | 設定例 | 説明 |
| :--- | :--- | :--- |
| config_file | `care_roujin-home.json` | 業種設定ファイル |
| address_csv | `default` | 地域データ |
| output_file | `default` | 出力ファイル名 |
| max_requests | `3` | リクエスト数/地域 |

**処理時間**: 1〜3時間（業種依存）

## FAQ

**Q1: レビュー監視と施設情報収集の違い**

| 項目 | レビュー監視 | 施設情報収集 |
| :--- | :--- | :--- |
| 取得データ | 口コミレビュー | 施設基本情報 |
| 前提条件 | 施設リスト保有 | なし |
| 対象業種 | 歯科、結婚相談所 | 介護施設 |

**Q2: 介護施設のレビュー取得手順**

1. 施設情報収集（`brightdata_facility.yml`）→ 施設データCSV作成
2. レビュー監視（`brightdata_reviews_新仕様.yml`）→ レビューCSV作成

**Q3: ワークフロー選択基準**

- 施設リスト保有 → `brightdata_reviews_新仕様.yml`
- 施設リスト未保有 → `brightdata_facility.yml`

**Q4: テスト実行の推奨設定**

| モード | 設定ファイル | 費用 | 時間 |
| :--- | :--- | :--- | :--- |
| レビュー監視 | `dental_test.json` | 約100円 | 5〜10分 |
| 施設情報収集 | `care_roujin-home_test.json` | 約$1 | 5〜10分 |
