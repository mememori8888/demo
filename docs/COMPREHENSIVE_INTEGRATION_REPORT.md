## 🎯 統合ルール

各処理フェーズで適用された統合ルール:

### 1. スターホワイトニング削除
- 施設名に「スターホワイトニング」を含む全データを削除

### 2. WordPress独自+外部データ統合
- **条件**: 同じ電話番号 + 類似施設名（類似度 ≥ 0.6）
- **保持**: WordPress独自のpost_id
- **採用**: 外部データの緯度経度、その他詳細情報

### 3. 同名・同電話番号統合
- **条件**: 同じ電話番号 + 正規化後の施設名が一致（類似度 ≥ 0.7）
- **保持**: 小さいpost_id
- **統合**: すべての情報を保持post_idにマージ

### 4. 最終重複統合
- **条件**: 同じ施設ID、または同じ電話番号・施設名・住所
- **保護**: 多店舗展開（緯度経度差 > 0.01度 ≈ 1km）
- **保持**: WordPress独自post_id優先、なければ小さいpost_id

### 5. 電話番号変更統合
- **条件**: 同じ施設名・住所、近い位置（緯度経度差 < 0.001度 ≈ 100m）、異なる電話番号
- **保持**: WordPress独自のpost_id
- **採用**: 施設ID有りの電話番号（より信頼性が高い）
