155回の手動実行は大変です。代わりに：

### オプション1: split_mode を使用（20分割で自動並列実行）

```
csv_file: results/dental_new.csv
workers: 7
skip_if_empty: web
split_mode: ✅ true
num_splits: 20
```

**1回の実行で全体が完了**（約3時間）

### オプション2: より大きなバッチサイズ（5000行ずつ、16回実行）

```
実行 1: start_row: 1 / end_row: 5000
実行 2: start_row: 5001 / end_row: 10000
実行 3: start_row: 10001 / end_row: 15000
...
実行 16: start_row: 75001 / end_row: 77346
```

各実行は約20-30分。
