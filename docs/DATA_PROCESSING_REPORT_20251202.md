## 6. 主要な問題と対応

### 6.1 正規化の課題
**問題**: 施設名の正規化で「都道府県市区町村-」プレフィックスが重複検出を阻害

**対応**: `normalize_facility_name()`を修正
```python
if '-' in name_str:
    parts = name_str.split('-', 1)
    if len(parts) == 2:
        name_str = parts[1]  # 施設名部分のみ抽出
```

### 6.2 住所の微妙な違い
**問題**: 「音更町柳町南区1-23」vs「音更町柳町１−２３」のように住所表記が微妙に異なるため完全一致しない

**対応**: 類似度マッチング（70%以上）で検出可能に

### 6.3 重複削除の優先順位
**問題**: 当初「電話番号+緯度経度」が最優先だったが、GoogleMapのみのデータの方が信頼性が高い

**対応**: 優先順位を変更
- 修正前: 電話番号+緯度経度 > wordpress検索結果
- 修正後: GoogleMapのみ > 電話番号+緯度経度 > wordpress検索結果
