# 閉業施設クリーンアップ処理レポート

**実施日**: 2025年12月1日  
**処理対象**: インテグレート納品スプレッドシート

---

## 📋 処理概要

WordPress独自データに含まれる閉業施設を特定し、データベースから削除しました。

---

## 🔍 処理内容

### 1. データ照合
- **対象データ**: `integrated_data_updated`シート内のWordPress独自データ（23,870件）
- **照合先**: `closed_facilities`シート（5,435件）

### 2. 照合方法
以下の条件で施設を照合し、一致した場合に閉業施設と判定：

- **電話番号**による照合（最優先）
- **正規化施設名 + 正規化住所**による照合
- 施設名、住所での補完照合

### 3. 検出結果
- **一致検出数**: 2,822件
  - 電話番号一致: 2,733件
  - 名称+住所一致: 89件
